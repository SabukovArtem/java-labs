## Описание заданий

### 1. Папка `10variant` – Учёт литературы (Вариант 10)
Реализована иерархия классов:
- Базовый класс `Literature` (код, тип, название, год, издательство, страницы, автор).
- Производные классы:
  - `ScientificTechnicalLiterature` (научно-техническая: область науки, количество экземпляров).
  - `Periodical` (периодика: вид, период издательства).
  - `ReferenceBook` (справочники: направление, том, часть).

Класс `Library` хранит коллекцию объектов.  
**Возможности:**
- Добавление литературы разных типов через консольное меню.
- Просмотр всего списка.
- Сортировка по названию и по году издательства.
- Сохранение данных в текстовый файл и загрузка из файла.

### 2. Папка `Arifmetic` – Базовые вычисления
- **Task1:** По двум числам a и b найти наибольшее max(a,b) и вычислить d = cos(max(a,b)).
- **Task2:** (аналогичная задача – уточните при необходимости).

### 3. Папка `Massive` – Операции с массивами
#### Task3 – Одномерный массив
Программа создаёт массив случайных чисел и выполняет 12 заданий (поиск индексов, замена min/max, фильтрация, уникальные значения и др.).

#### Task4 – Двумерный массив (матрица)
12 операций над матрицей: подсчёты по строкам и столбцам, минимумы/максимумы, проверка упорядоченности и т.д.

### 4. Папка `Zadacha` – Аргументы командной строки
**Task10:** Программа принимает два числа `x` и `y` как аргументы командной строки, находит меньшее из них и делит его на 2. Результат выводится на экран.

## Инструкция по запуску

Все программы компилируются стандартным компилятором `javac`. Убедитесь, что Java установлена корректно (`java -version`).

### Общий порядок действий
1. Перейдите в папку с исходным кодом нужного задания (например, `cd Massive/src`).
2. Скомпилируйте все `.java` файлы в папке:
   ```bash
   javac *.java
Запустите нужный класс:

bash
java ИмяКласса
Для Task10 обязательно укажите два числа:

bash
java Task10 5 3
Запуск в IntelliJ IDEA
Если вы используете IntelliJ IDEA, проекты уже настроены. Для запуска Task10 с аргументами:

Откройте Run → Edit Configurations.

В поле Program arguments введите два числа, например 8 3.

Нажмите OK и запустите программу.

Примечания
Во всех программах, кроме Task10, ввод данных осуществляется с клавиатуры (через Scanner).

В задании с литературой (10variant) после запуска появляется интерактивное меню.

Исходный код содержит комментарии к ключевым моментам.
